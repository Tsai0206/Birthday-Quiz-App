# 背景音樂設置

## 如何添加 Yuno Miles 音樂

1. 從 YouTube 或其他來源下載 Yuno Miles 的音樂
   - 推薦使用 YouTube to MP3 轉換器
   - 選擇任何你喜歡的 Yuno Miles 歌曲

2. 將音樂文件重命名為 `yuno-miles.mp3`

3. 將文件放在這個資料夾中：
   ```
   public/music/yuno-miles.mp3
   ```

4. 重新啟動開發伺服器或重新部署

## 可選的音樂文件格式
- MP3 (推薦)
- WAV
- OGG

## 注意事項
- 確保音樂文件大小合理（建議 < 10MB）
- 音樂會自動循環播放
- 預設音量為 30%
- 右下角有播放/暫停和靜音按鈕

## Yuno Miles 推薦曲目
你可以選擇以下任何一首歌或整個 playlist：
- "Road to Riches"
- "Swervin"
- "Stop Lyin"
- 或任何你喜歡的 Yuno Miles 作品！
